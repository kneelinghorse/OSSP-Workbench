# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-14

### Added
- npm distribution metadata for publishing and installation.
- CLI binary entrypoint `ossp-consumer-gen` for consumer generation.
- GitHub Actions workflow for npm publish on `v*` tags.
- Packaging test coverage for publish metadata and `npm pack` contents.

### Changed
- Package version advanced to `1.0.0` for first stable npm release.
- Consumer generation command wiring updated to ESM-native imports.
